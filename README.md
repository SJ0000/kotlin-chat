# Simple Messenger Server
웹 브라우저 환경에서 동작하는 메신저 어플리케이션 서버

### 사용 기술
- Kotlin
- Spring boot
- QueryDsl
- Redis
- MongoDB

### 주요 기능
- Websocket, STOMP 프로토콜을 사용한 실시간 채팅
- JWT 토큰 인증 방식을 사용한 로그인
- 친구 기능 (친구 추가 요청 및 친구 목록 관리)
